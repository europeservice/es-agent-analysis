11.11 — CashFlow та схема руху коштів

Фінансист Гриценко Марина будує систему CashFlow у Finmap. Підключені всі рахунки. Після завершення налаштування — повна картина надходжень та витрат з розбивкою по напрямках.

Схема руху коштів:

ДОХОДИ ФІЛІЙ (від клієнтів → ФОП 2-ї групи) ФОП Тестов → ЗП менеджерів ЦО + Звягель ФОП Курасов → ЗП менеджерів Одеси ФОП Лисун → ЗП менеджерів Рівного + 1/3 прибутку керівнику ФОП Литвин → ЗП Житомир + оренда Житомир/Звягель (дивіденди) Якщо не вистачає → покриття різниці з ТОВ ДОХОДИ ЦЕНТРУ (від партнерів у валюті → ТОВ / ФОП Котов) → ЗП штату (адмін + маркетинг) → Реклама (Google ~150к, FB ~40к, Jooble ~83к, Work.ua ~46к...) → Оренда Київ ЦО (~50к) → Підрядники (COO, Google, SEO, FB таргет) → Податки (ЄП 6% від доходу ТОВ/ФОП 3) → Інші операційні витрати
